Airbnb Price Optimizer

This was a project that I worked on with three other web development students and five data science students. It allows Berlin based Airbnb hosts to get an optimal price for their listings using a machine learning algorithm that was trained on data from a database of previously successful stays. My contribution was the landing/marketing page and the about page.

Currency Records

This is a project that I completed individually. It is a website that uses European Central Bank currency records from https://exchangeratesapi.io/ to provide users with the ability to request daily exchange rates between major world currencies for any day since the year 2000.

Github User Profile

This is a project that I completed individually. It pulls profile data from the Github api about an individual Github user including name, avatar, location, and followers. The user can then search for another profile by username utilizing the provided search input.
